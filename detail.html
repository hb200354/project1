※ SQL 처리 흐름 요약

1. 사용자가 상품 목록에서 해당 상품 클릭
   → detail.html?id=1 이동

2. DB에서 해당 상품 조회
   SELECT * FROM products WHERE id = 1;

3. 결과를 받아 HTML로 출력
   → 상품명, 가격, 설명, 이미지 표시
